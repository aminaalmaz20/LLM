Комментарий к доработке автотестов: дополнительно к имевшемуся у нас промпту для созания автотестов из учебника, я попросила модель дописать тесты по следующим сценариям:
1. Валидация входных данных:
- Пустая строка → ожидаем ValueError или HTTP 400
- Строка из пробелов
- Очень длинный текст (>5000 символов)
- Неподдерживаемый язык

2. Проверка структуры запроса к API:
- Убедись, что в запросе:
  - используется правильный model_name
  - prompt содержит исходный текст
  - headers содержат Authorization и Content-Type

3. Некорректные ответы API:
- status 200, но response отсутствует
- status 200, response = null
- status 429 (rate limit)
- Timeout exception
